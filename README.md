# Vellum

A spatial-first workspace; a modern alternative to Notion built with Next.js and Tailwind CSS.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Or connect your GitHub repo at [vercel.com/new](https://vercel.com/new) for automatic deployments on every push.
