# Vellum

> A spatial-first workspace. A modern, minimal alternative to Notion.

**[Live Demo →](https://vellum-indol.vercel.app)**

---

## Overview

Vellum is a spatial workspace app built with Next.js 14 and TypeScript. It reimagines how teams organise information — moving away from hierarchical documents toward a canvas-based, block-driven interface with real-time-feel interactions.

---

## Features

### Sidebar Navigation
- Organised Spaces (Engineering, Design, Growth)
- Recent documents and Favorites
- One-click Space creation

###  Block System
- **Text Block** — Rich text editing
- **Card Block** — Structured block with header and metadata
- **Database Block** — Table, board, or timeline views
- Drag-to-arrange canvas layout

### Command Palette (`⌘K`)
Full keyboard-driven navigation including:
- Jump to Space (`⌘G`)
- Full-text search across all workspaces (`⌘F`)
- Copy block link (`⌘L`)
- Toggle dark mode (`⌘⇧L`)
- Create blocks via keyboard shortcuts (T, C, D)

### Dark Mode
System-aware with manual toggle via command palette.

### Collaboration UI
- Multi-user avatars on documents
- Task status indicators (active, in review, done)
- Priority flags and due dates
- Progress tracking per block
